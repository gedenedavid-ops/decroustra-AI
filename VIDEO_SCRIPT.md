# SCRIPT VIDEO — decroustra.AI (2 minutes)

## 0:00-0:25 | LE PROBLÈME
[Capture d'écran Facebook avec des posts éparpillés]

En Côte d'Ivoire, quand une personne disparaît, les familles publient sur Facebook.
Mais les annonces sont éparpillées sur des dizaines de groupes.
Impossible de tout suivre. Aucune centralisation n'existe.

## 0:25-0:50 | LA SOLUTION
[Interface decroustra.AI qui s'ouvre]

decroustra.AI résout ce problème.
Un agent IA qui scrute en continu les groupes Facebook publics,
extrait automatiquement les disparitions, et permet aux familles
de chercher en langage naturel.

## 0:50-1:20 | DÉMO LIVE
[Capture de l'interface — taper des questions]

"Disparitions à Yopougon ?"
→ 5 alertes trouvées, avec noms, âges, contacts.

"Enfants disparus ?"
→ 16 enfants de 2 à 17 ans.

"Je recherche mon fils David, 11 ans, Abidjan ?"
→ 10 alertes à Abidjan, avec proposition d'affiner.

## 1:20-1:40 | BRIGHT DATA
[Schéma technique ou code]

Techniquement, on utilise Bright Data :
- Web Unlocker pour contourner les blocages Facebook
- Facebook Scraper API pour les posts structurés
- SERP API pour la recherche Google

7 139 posts scrappés → 1 872 alertes extraites avec DeepSeek.

## 1:40-1:50 | IMPACT
[Interface avec les chiffres]

- 1 872 alertes centralisées
- 949 localités couvertes
- Dédoublonnage automatique
- Gratuit pour les familles

## 1:50-2:00 | TRACK 3 CONCLUSION
[Logo + Track]

Track 3 : Security & Compliance.
Parce que l'information, c'est la sécurité des personnes.
decroustra.AI — Centralisons les alertes. Retrouvons les personnes.
